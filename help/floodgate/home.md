# Upptäck Adobe Target Floodgate {#get-started}


## Vad är Floodgate?

Floodgate är ett system för funktionshantering och experimenterande som hjälper applikationer att leverera nya upplevelser säkert, gradvis och med mindre risk.

I stället för att aktivera en ny funktion för alla samtidigt kan man låta teamen

* Frigör funktioner gradvis
Börja med en liten andel användare och öka sedan med tiden. Om något inte beter sig som väntat kan funktionen stängas av snabbt, utan driftavbrott eller ominstallation av programvara.

* Målgrupper för specifika målgrupper
Aktivera nya funktioner för valda grupper (till exempel betaprogram, vissa regioner eller specifika planer) så att ändringarna kan valideras och förfinas innan de blir allmänt tillgängliga.

* Kör experiment och A/B-tester
Testa olika versioner av en funktion (t.ex. alternativa layouter, flöden eller rekommendationer) och mät vilken som fungerar bäst med data som underlag för produktbesluten.

* Minska risken för viktiga releaser
Stora förändringar kan delas upp i mindre, kontrollerade rullningar, så det är lättare att upptäcka, förstå och åtgärda problem innan de påverkar alla.

* Snabba svar på problem
Om ett fel eller prestandaproblem uppstår kan team bara inaktivera den funktion som påverkas, vilket gör resten av programmet stabil.

I praktiken gör Floodgate det enklare för slutanvändarna att uppleva fler förbättringar med färre avbrott: nya funktioner kan komma fram tidigare, testas mer noggrant och justeras snabbt baserat på feedback från verkligheten.
