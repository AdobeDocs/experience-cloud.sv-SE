# Kontakta support {#contact-support}

Om du behöver hjälp av vårt supportteam för upplevelselanseringar kan du skicka en förfrågan via Adobe standardsupportkanal. Supportteamet övervakar inkommande förfrågningar och svarar så snabbt som möjligt.

## Skicka in en supportförfrågan {#submit-request}

Så här skickar du en supportförfrågan:

1. Logga in på [Adobe Admin Console](https://adminconsole.adobe.com).
2. Navigera till **Support** i den övre navigeringen.
3. Välj **Öppna en supportanmälan**.
4. Fyll i begärandeformuläret med information om ditt problem.
5. Tilldela biljetten till **Experience Rollouts**-produkten och skicka.

Du kan också använda alternativet **Kontakta support** som finns på den högra panelen i Experience Rollouts-konsolen för att öppna en supportförfrågan direkt från produkten.

## Vad som ska ingå i din förfrågan {#what-to-include}

Om du vill hjälpa supportteamet att undersöka ditt problem så effektivt som möjligt kan du inkludera följande information i din supportförfrågan:

* **Miljö** - Ange om problemet inträffar i Stage, Production eller båda.
* **Funktionsflagga eller release** - Ange namn, nyckel eller direktlänk till den berörda funktionsflaggan, funktionsgruppen eller releasen.
* **Testa användardata** - Inkludera användaridentifieraren (e-post, GUID eller besökar-ID) för en användare som har problem.
* **Steg för att återskapa** - Beskriv exakt vilka steg du tog som ledde till problemet.
* **Förväntat jämfört med faktiskt beteende** - Beskriv vad du förväntade dig och vad som inträffade i stället.
* **Skärmbilder eller felmeddelanden** - Bifoga skärmdumpar, felmeddelanden eller API-svar som illustrerar problemet.
* **Annan relevant kontext** - inkludera SDK-version, program-klient-ID eller nyligen gjorda konfigurationsändringar som kan vara relaterade.

Ju mer detaljerad du anger, desto snabbare kan supportteamet diagnostisera och lösa problemet.
