# Hantera program {#manage-applications}

Ett **program** i Experience Rollouts representerar tjänsten eller produkten som du vill kontrollera med funktionsflaggor. Innan du kan skapa funktionsflaggor måste du ta med minst ett program till konsolen.

## Vem kan hantera program? {#permissions}

Endast medlemmar med rollen **Admin** kan lägga till eller ändra program. Kontakta administratören om du inte har den här rollen.

## Lägga till ett program {#add-application}

Information om hur du lägger till ett program finns i [Anpassa programmet](onboard-your-application.md) för steg-instruktioner.
